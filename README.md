#  Проект для конкурса "Большие вызовы"

Прогнозирование интенсивности трафика с использованием LSTM-нейросети.

##  Быстрый старт

1. Установка: `pip install -r requirements.txt`
2. Генерация данных: `python src/generate_sample_data.py`
3. Обучение: `python src/traffic_forecast.py`
4. Прогноз: `python src/traffic_forecast.py --predict`

## Как это работает

Простыми словами:

1. Нейросеть изучает историю — анализирует данные за последние 24 часа (как менялся трафик)

2. Находит закономерности — понимает паттерны:
   - Утром (8:00-9:00) — час пик, много машин
   - Днём — спад трафика
   - Вечером (17:00-19:00) — снова час пик
   - Ночью — минимум трафика

3. Запоминает через LSTM — это специальный тип нейросети, которая:
   - Имеет память о прошлых значениях
   - Понимает последовательность (что было раньше, что сейчас)
   - Учитывает тренды (растёт трафик или падает)

4. Делает прогноз — на основе изученного предсказывает, сколько машин будет в следующий час

Архитектура:

Вход (24 часа) → LSTM(64) → LSTM(32) → Dense → Прогноз

- 2 слоя LSTM — извлекают временные зависимости
- Dropout (20%) — защита от переобучения
- Точность: ~90% (ошибка в среднем 5.3 машины)
