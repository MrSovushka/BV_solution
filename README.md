#  Traffic LSTM Forecast

Прогнозирование интенсивности трафика с использованием LSTM-нейросети.

##  Быстрый старт

1. Установка: `pip install -r requirements.txt`
2. Генерация данных: `python src/generate_sample_data.py`
3. Обучение: `python src/traffic_forecast.py`
4. Прогноз: `python src/traffic_forecast.py --predict`

##  Метрики
- MAE: 5.3 машины
- Точность: ~90%
